# 🌱 Plant Nursery Web Application

## 🚀 Project Overview

A full-stack plant nursery web app where users can browse plants, view details, and interact with plant data fetched from an external API.

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* JavaScript
* HTML
* CSS
* Tailwind CSS

### Backend:

* Node.js
* Express.js
* MongoDB

### API Used:

* Trefle Plants API (for plant & seed data)

---

## ✨ Features

* 🌿 Browse plants, seeds and garden accessories
* 🔍 Search and view plant details
* 🛒 Add to cart functionality
*     Order placement system  
- 🧑‍💼 Admin dashboard (UI)  
- 📊 Manage orders (Admin panel)  
- 🔐 Backend integration with MongoDB  
- 📱 Fully responsive UI (Tailwind CSS)

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/PlantNursery.git
```

### 2. Install dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### 3. Create environment file

Create `.env` inside `server` folder:

```env
PORT=5000
MONGO_URI=your_mongodb_url
API_KEY=your_trefle_api_key
```

### 4. Run the project

```bash
# Run backend
npm run server

# Run frontend
npm start
```
